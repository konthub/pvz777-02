Блок 1

Шаг 1. Расчет удержаний

Видео
1. Открыть раздел расчёта удержаний

После нажатия на кнопку «Расчёт зарплаты» откроется таблица с инструментами для расчёта.

Первый шаг в этом процессе — расчёт удержаний.

Нужно нажать кнопку «Расчёт удержаний».

2. Найти нужный ПВЗ

В открывшейся таблице есть список всех ПВЗ.

У каждого ПВЗ есть своя отдельная ссылка.

Важно:
один ПВЗ = одна ссылка.

Чтобы быстро найти нужный адрес:

Нажмите Ctrl + F.
Начните писать адрес по-русски.
Система автоматически найдёт нужную строку.
Откройте ссылку, которая стоит напротив нужного адреса.
3. Заполнить основные данные в файле ПВЗ

После открытия файла нужно заполнить верхнюю часть документа.

Обязательно указывается:

Что заполнить Как заполнять
Дата Поставить дату, к которой относится запись
Адрес ПВЗ Выбрать адрес из общего списка
Управляющий Выбрать себя из списка управляющих

Важно: адрес в файле должен совпадать с адресом ПВЗ, по которому вы работаете.

4. Работать с блоками WB / Ozon / Яндекс

В файле есть три одинаковых блока:

WB
Ozon
Яндекс

Эти блоки нужны для фиксации:

зависших позиций;
браков;
подмен;
утерь;
возвратов;
штрафов;
украденных товаров.

Если проблема относится к WB — заполняете блок WB.
Если к Ozon — блок Ozon.
Если к Яндексу — блок Яндекс.

5. Вносить записи по каждой позиции

Одна строка в таблице — это одна отдельная запись из системы маркетплейса.

По каждой позиции нужно внести данные:

Поле Что указать
ШК Штрихкод позиции
Название Название товара
Сумма Сумма по позиции
Статус Что произошло: подмена, брак, зависший, утеряние, возврат, штраф или украли
Сотрудник К какому сотруднику относится ситуация
Решение Что делать с этой суммой
Комментарий При необходимости написать пояснение
6. Выбрать сотрудника

Сотрудника нужно выбирать из списка.

Как это сделать:

Откройте поле с сотрудником.
Начните писать фамилию.
Система сама подтянет сотрудника из списка.
Выберите нужного сотрудника.

Важно: вручную полностью вносить сотрудника не нужно.

7. Обязательно выбрать решение

После внесения записи нужно выбрать решение.

Возможные варианты:

Решение Когда используется
Оспорено Если ситуация оспорена
Нельзя оспорить Если оспаривание невозможно
Для ПВЗ Если ситуация относится к ПВЗ
Списать с зарплаты сотрудника Если принято решение удержать сумму с сотрудника

Самое важное:
если нужно, чтобы сумма попала в расчёт удержаний, должно быть выбрано решение «Списать с зарплаты сотрудника».

8. Проверить полноту заполнения

Чтобы данные корректно попали в сводную таблицу, по каждой строке обязательно должны быть заполнены:

Фамилия сотрудника.
Решение.

Без этих данных сумма может не попасть в итоговый расчёт.

02 Сводный файл

Видео
1.. Как работает сводный файл

Все данные из файлов ПВЗ автоматически собираются в общий сводный файл.

Данные подтягиваются из всех файлов ПВЗ, поэтому вручную суммировать ничего не нужно.

В сводном файле можно посмотреть:

Раздел Что показывает
Данные по всем ПВЗ Общая информация по всем адресам
Расчёт по сотрудникам Сумма к удержанию по сотруднику, адрес ПВЗ, управляющий и количество случаев
Сводная по сотруднику Итоговая сумма по каждому сотруднику

2. Главное правило работы

В своём файле ПВЗ нужно аккуратно заносить всю информацию по каждому случаю.

Особенно важно:

не забывать выбирать сотрудника;
не забывать выбирать решение;
указывать корректную сумму;
работать именно в файле своего ПВЗ;
при необходимости оставлять комментарий.
