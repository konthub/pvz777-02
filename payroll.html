📋 Инструкция: Расчёт заработной платы (ПВЗ)
1. Источники данных
Для расчёта используются:
1. 📊 Реестр (ежедневный)
o зависшие
o браки
o подмены
2. 📋 Табель
o берётся из того же реестра
3. 📥 Выгрузка от Нади
o приходит 1 числа (после 10-го дня периода)
o содержит полный отчет по ПВЗ
________________________________________
2. Подготовка данных
1. Из реестра:
o делаем выжимку
o переносим в Excel-файл
2. Переносим:
o зависшие
o браки
o подмены
________________________________________
3. Финансовый расчёт (уровень ПВЗ)
В фин-чате рассчитываем:
• оборот (сколько должны были получить)
• фактические поступления
• скорость приёмки
• рейтинг ПВЗ
• мотивацию за переупаковку
• зависшие / браки / подмены
________________________________________
4. Расчёт по сотрудникам
На основании реестра:
1. Определяем по каждому сотруднику:
o недостачи
o ответственность за операции
2. Вносим в зарплатную ведомость:
o рейтинг
o скорость
o переупаковка
o начисления
o авансы
📌 В итоге:
• получаем сумму к выплате
________________________________________
5. Формирование итогового файла
• Все расчёты:
o собираются в единый файл
• Файл:
o сохраняется
o передаётся на проверку
________________________________________
6. Проверка бухгалтерией
Файл отправляется Марине
Она проверяет:
• корректность расчётов
• соответствие финансовому отчёту
• совпадение данных
________________________________________
7. Контроль корректности
Обязательная проверка:
• сумма зависших в фин. отчёте
• сумма зависших в реестре
📌 Особенность:
• расхождения чаще всего по зависшим
• браки и подмены обычно совпадают 1:1
________________________________________
8. Действия при расхождениях
Если есть несоответствия:
1. Запросить доступ у Нади
2. Зайти в баланс
3. Проверить:
o на какой день есть расхождение
4. Пересчитать
________________________________________
9. Финальная проверка перед отправкой
Перед отправкой убедиться:
• все зависшие указаны
• все браки и подмены учтены
• назначены виновные
• есть объяснения по каждому случаю
• рассчитаны:
o рейтинг
o скорость
o переупаковка
• учтены:
o авансы
o зарплаты (включая официальные)
________________________________________
10. Завершение процесса
1. Отправка Марине
2. Быстрая проверка
3. Готовность расчёта — в течение 1 дня
