Открытие дня на ПВЗ

Видео
📋 Инструкция: Открытие дня на ПВЗ
1. Открытие точки
1. Подойти к ПВЗ
2. Снять сигнализацию
3. Открыть дверь
4. Зайти внутрь
________________________________________
2. Подготовка к работе
1. Переодеться
2. Включить компьютер
3. Подготовить рабочее место
📌 Обычно при открытии:
• уже есть ночная приёмка (много коробок)
________________________________________
3. Подготовка зоны приёмки
👉 Важно:
• рабочий стол должен быть чистым
• без посторонних предметов
________________________________________
4. Начало приёмки
1. Открыть режим:
o «Приёмка»
2. Открыть коробку
3. Выложить товары на стол
📌 Рекомендуется:
• выкладывать на стол, а не работать из коробки
________________________________________
5. Процесс сканирования
1. Берём товар
2. Сканируем штрих-код
3. Система:
o показывает ячейку хранения
4. Маркером:
o пишем номер ячейки на товаре
📌 Работать до появления:
• номера ячейки на экране
________________________________________
6. Работа в команде
Если 2 человека:
• 1-й:
o сканирует
• 2-й:
o сразу раскладывает по ячейкам
📌 Особенно актуально:
• при большом потоке (несколько ПВЗ)
________________________________________
7. Приоритет приёмки
👉 В первую очередь принимаем:
• габаритные товары
• тяжёлые коробки
📌 Причина:
• освобождаем пространство
• уменьшаем хаос
________________________________________
8. Работа с курьером
При приёмке:
• просить курьера:
o ставить товар штрих-кодом вверх
📌 Это ускоряет работу
________________________________________
9. Выявление брака при приёмке
Если товар повреждён:
1. Сканируем:
o сначала код брака
2. Затем:
o сканируем товар
📌 Важно:
• зафиксировать до выдачи клиенту
• камера должна это записать
________________________________________
10. Излишки (чужие товары)
Если при сканировании:
• система показывает «излишек»
👉 Действие:
1. Добавить в коробку возврата
2. Отправить обратно
📌 Переупаковка:
• не требуется (по старому процессу)
• сейчас может обновляться
________________________________________
11. Ошибки при приёмке
Частые ошибки:
1. ❌ Сканируют не тот штрих-код
2. ❌ Путают ячейки (например: 339 → 239)
3. ❌ Берут возвратную коробку как новую поставку
4. ❌ Работают в спешке
📌 Важно:
• система ошибки не пропустит
• но человеческий фактор остаётся
________________________________________
12. Работа с возвратными коробками
⚠️ Важно:
• не путать:
o поставку
o возврат
📌 Ошибка:
• вскрыли возвратную коробку
→ товар снова попадёт в приёмку
________________________________________
13. Нестандартные ситуации
Ситуация:
• товар числится выданным клиенту
• но физически лежит на ПВЗ
👉 Действие:
• временно разместить отдельно
• дождаться клиента
📌 Часто бывает у новичков
________________________________________
14. Завершение приёмки
• все товары:
o отсканированы
o разложены по ячейкам
• зона:
o приведена в порядок

Работа с браком

Видео
📋 Инструкция: Работа с браком (WB, ПВЗ)
1. Общий принцип
👉 Главное правило:
Любой дефект → сразу ставим “Брак”
📌 Почему это критично:
• если НЕ отметить брак → будет удержание
• если отметить → можно оспорить через камеры
________________________________________
2. Осмотр товара (одежда)
При проверке (пример — куртка):
Проверяем:
• чистоту (грязь, пятна)
• подмышки (часто дефекты)
• карманы
• фурнитуру
• замки (обязательно проверить работу)
📌 Важно:
• белые вещи проверять особенно тщательно (пятна видны лучше)
________________________________________
3. Осмотр обуви
Проверяем:
• левый и правый ботинок
• одинаковый размер
• внешний вид:
o грязь
o царапины
o клей
• соответствие карточке товара
📌 Если есть клей / дефекты → сразу брак
________________________________________
4. Осмотр товара после примерки
Ситуация:
• клиент мерил
• сломал / повредил (например, заклёпку)
👉 Это:
• ответственность клиента
________________________________________
5. Товары, которые нельзя вскрывать
Нельзя вскрывать:
• духи
• косметику
• наушники
• товары в заводской упаковке
📌 Исключение:
• только по специальной заявке
________________________________________
6. Подмена (самый критичный риск)
Подмена = самый опасный случай
📌 Если не отметить:
• удержат полную стоимость товара
Признаки:
• товар не соответствует карточке
• другая модель / цвет / материал
👉 Действие:
• сразу ставим брак
________________________________________
7. Просрочка проверки (ошибка)
Ситуация:
• товар лежал 14 дней без проверки
👉 Риск:
• склад выставит удержание
• ответственность на сотруднике
📌 Вывод:
• проверка должна быть своевременной
________________________________________
8. Как поставить брак (процесс)
Шаги:
1. Открыть:
o «Вернуть товар»
2. Выбрать:
o «Да, вернуть»
3. Сканировать стикер
4. Выбрать причину:
o некачественная фурнитура
o повреждение
o грязь
o клей
o дефекты ткани
________________________________________
9. Фотофиксация брака
Обязательно сделать фото:
• товар спереди
• товар сзади
• сам дефект
• штрих-код (стикер)
📌 Если дефектов много:
• всё описываем в комментарии
________________________________________
10. Комментарий
Обязательно написать:
Пример:
• клей
• царапины
• грязь
📌 Даже если фото ограничены — текст обязателен
________________________________________
11. Завершение оформления
1. Отправить брак в системе
2. Товар:
o НЕ перепаковывается
o уходит в том же виде
📌 Это принципиально важно

Работа с подменой

Видео
📋 Инструкция: Работа с подменой (WB, ПВЗ)
1. Что такое подмена
👉 Подмена — это когда:
• товар не соответствует карточке
Примеры:
• другая модель
• другой цвет
• другой материал
• вообще другой товар
📌 Даже небольшое отличие = подмена
________________________________________
2. Главный риск
👉 Подмена = максимальное удержание
• если НЕ отметить →
❌ списывается полная стоимость товара
________________________________________
3. Как выявлять подмену
При проверке товара:
Сравниваем с карточкой:
• модель
• цвет
• размер
• материал
• внешний вид
📌 Пример:
• должны быть кожаные → пришли другие
→ это подмена
________________________________________
4. Действие при обнаружении
👉 СРАЗУ:
1. Отмечаем в системе:
o «Брак» (отдельной графы “подмена” нет)
2. Фиксируем:
o как дефект
________________________________________
5. Фотофиксация
Обязательно сделать фото:
• товар полностью
• несоответствие карточке
• детали (где видно отличие)
📌 Лучше:
• показать, что товар вообще другой
________________________________________
6. Важный момент (реальность работы)
Даже если:
• всё сделали правильно
• отметили брак
• сделали фото
👉 WB может:
• прислать удержание
• запросить видео
📌 Это нормальная практика
________________________________________
7. Как защищаться
Чтобы оспорить:
1. Должно быть:
o фото в системе
o отметка брака
o фиксация под камерой
2. Тогда:
o управляющий может оспорить
o удержание снимается
________________________________________

8. Где хранятся доказательства
📌 Важно:
• фото НЕ хранятся в телефоне
• всё хранится:
o внутри системы

Возвраты

Видео
📋 Инструкция: Возвраты WB (ПВЗ)
1. Общий принцип
👉 Каждый день появляются возвраты:
• товары, которые клиент не забрал за 14 дней
📌 Важно:
• обработка должна быть ежедневно утром
• иначе → риск удержаний
________________________________________
2. Ежедневная работа с возвратами
Каждое утро:
1. Заходим в раздел:
o «Возвраты»
2. По каждому возврату:
o смотрим ячейку
o записываем:
 номер ячейки
 последние цифры ШК
📌 Делается вручную (на листочек) — чтобы не запутаться
________________________________________
3. Поиск товара на складе
1. Идём на склад
2. По записанным данным:
o находим ячейку
o сверяем товар
________________________________________
4. Обязательная проверка товара
👉 КЛЮЧЕВОЕ ПРАВИЛО:
Каждый возврат проверяется перед отправкой
Даже если:
• товар лежал 14 дней
• никто его не трогал
📌 Почему:
• склад может выставить:
o брак
o подмену
→ удержание будет на ПВЗ
________________________________________
5. Как проверять возврат
Проверка проводится:
• под камерой
Проверяем:
• соответствие карточке товара
• размер
• цвет
• модель
• состояние:
o грязь
o дефекты
________________________________________
6. Исключение — заводская упаковка
👉 НЕ вскрываем:
• товары в заводской упаковке
📌 Все остальные:
• вскрываются обязательно
________________________________________
7. Подмена и брак
Если выявлено:
Подмена:
• товар не соответствует карточке
Брак:
• дефекты, повреждения
👉 Действие:
• фиксируем под камерой
• оформляем как брак
________________________________________
8. Первый этап — сбор товаров
1. Все проверенные товары:
o складываются во временную коробку
2. Это:
o буфер (чтобы не запутаться)
📌 Рекомендуется разделять:
• проверка
• переупаковка
________________________________________
9. Второй этап — переупаковка
Для каждого товара:
1. Сканируем
2. Система предлагает:
o тип пакета
Типы пакетов:
• маленький
• средний
• большой
📌 Важно:
• использовать правильный размер
• ошибка → штраф ~2500 ₽
________________________________________
10. Работа со стикерами
Есть 2 типа:
1. Стикер товара
• клеится на пакет
2. Стикер коробки (VT)
• используется для отправки возвратов
📌 Отличие:
• VT = возврат
________________________________________
11. Создание коробки
1. В системе:
o «Создать коробку»
2. Сканируем:
o стикер коробки (VT)
3. Клеим:
o на коробку
📌 Важно:
• клеить ниже → чтобы не заклеили скотчем
• лишние штрих-коды зачеркивать
________________________________________
12. Заполнение коробки
1. Складываем товары в коробку
2. После заполнения:
o закрываем коробку в системе
o физически заклеиваем
________________________________________
13. Завершение
• коробка готова
• передаётся курьеру
